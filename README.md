# Simulador de Investimento em C

Este código em C calcula o valor final de um investimento utilizando **juros simples**.  

## Como funciona

O usuário informa:
- Valor inicial do investimento  
- Número de anos  

O programa retorna o valor final do investimento após o período informado, considerando uma taxa fixa de **5% ao ano**.

## Recursos adicionais

O código também demonstra:
- Uso de **vetores e matrizes**  
- Manipulação de **ponteiros**  
- Exibição de **endereços de memória** no terminal
